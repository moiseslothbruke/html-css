O h1 é um exemplo de BOX-LEVEL

E os parágrafos também são um bom exemplo de caixas BOX-LEVEL já OS LINKS são um exemplo de caixas INLINE-LEVEL.

Exemplo de INLINE-LEVEL ESTE LINK É UM ELEMENTO INLINE LEVEL.

Ao verificar as configurações das caixas no DevTools é possível notar que existem configurações CSS padrões do user agent(O navegador) para cada caixa.
Verificando o parâmetro display pode-se observar que toda caixa BOX-LEVEL tem como argumento display: block já as caixa INLINE-LEVEL tem o argumento inline-block
